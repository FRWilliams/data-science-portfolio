# Experimentation & A/B Testing

This folder contains projects focused on **experiment design, analysis, and decision-making**, a core skill for Data Scientist II roles.

Projects emphasize:
- Proper experimental design
- Statistical rigor
- Business-aligned decision rules
- Understanding failure modes

### Topics Covered
- A/B testing frameworks
- Power and sample size analysis
- Hypothesis testing
- Guardrail metrics
- Sequential and observational experiments

### Typical Tools
- Python statistical libraries
- SciPy / Statsmodels
- Pandas
- Visualization for experiment interpretation

These projects demonstrate how to move from experiment results to **confident business recommendations**.
