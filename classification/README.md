# Classification Projects

This folder contains applied machine learning projects focused on **supervised classification problems** commonly encountered in industry.

The emphasis is on:
- Translating business problems into classification tasks
- Handling class imbalance
- Selecting appropriate evaluation metrics
- Interpreting models and tradeoffs
- Connecting predictions to real-world decisions

### Topics Covered
- Customer churn prediction
- Fraud and risk classification
- Binary and multiclass classification
- Feature engineering and selection
- Model comparison and threshold tuning
- Explainability (e.g., SHAP)

### Typical Tools
- Scikit-learn
- XGBoost / Random Forest
- Pandas / NumPy
- SHAP
- SMOTE and imbalance handling techniques

Each project is structured as an end-to-end workflow, not a single notebook.
